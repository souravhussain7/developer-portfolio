Front-end Web developer

A web developer is a programmer who specializes in, or is specifically engaged in, the development of World Wide Web applications using a client-server model. The applications typically use HTML, CSS and JavaScript in the client, PHP, ASP.NET (C#), Python, Node.js, Go or Java in the server, and http for communications between client and server.There are no formal educational or license requirements to become a web developer
